BootStrap

BootStrap is a more advanced program that we used in this project. BootStrap has the ability to make a webpage "mobile friendly" which means that any webpage that utilizes BootStrap correctly is compatible with mobile devices. This means that this webpage can be viewed on a mobile device and it will be resized to be compatible with the screen size, resolution, and other aspects
